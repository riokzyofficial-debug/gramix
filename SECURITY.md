# Security Policy

## Supported Versions

Only the latest release of gramix receives security fixes.

| Version | Supported |
|---|---|
| 0.1.x (latest) | ✅ |
| < 0.1.0 | ❌ |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a vulnerability, open a [GitHub Security Advisory](https://github.com/riokzyofficial-debug/gramix/security/advisories/new) in this repository. You will receive a response within 72 hours.

Please include as much of the following information as possible:

- A description of the vulnerability and its potential impact
- The affected component (e.g. `bot.py`, `fsm.py`)
- Steps to reproduce or a proof-of-concept
- Any suggested fix, if you have one

## Disclosure Policy

Once a fix is confirmed and released, the vulnerability will be disclosed publicly via a GitHub Security Advisory and noted in the [CHANGELOG](https://github.com/riokzyofficial-debug/gramix/blob/main/CHANGELOG.md).
